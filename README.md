# PRODIGY_DS_Task02

## 📌 Task Description
Perform data cleaning and exploratory data analysis (EDA) on the Titanic dataset (or provided sample dataset).  
Explore the relationships between variables and identify patterns and trends in the data.

## 📂 Dataset
- Source: [Prodigy InfoTech Data Science Datasets](https://github.com/Prodigy-InfoTech/data-science-datasets/tree/main/Task%202)  
- Alternative: [Kaggle Titanic Dataset](https://www.kaggle.com/c/titanic/data)

## 🛠️ Tools & Libraries
- Python (Pandas, NumPy, Matplotlib, Seaborn)  
- Power BI (Dashboard Visualization)  
- Jupyter Notebook / Google Colab  

## 🔎 Data Cleaning
- Handled missing values in `Age`, `Cabin`, and `Embarked`  
- Converted categorical variables (`Sex`, `Embarked`, `Survived`)  
- Removed duplicates and corrected data types  

## 📊 Exploratory Data Analysis (EDA)
- **Age Distribution** → Histogram  
- **Survival by Gender** → Stacked Bar Chart  
- **Survival by Class** → Clustered Bar Chart  
- **Embarked Distribution** → Pie Chart (Southampton, Cherbourg, Queenstown)  

## 📈 Insights
- Younger passengers had higher survival chances  
- Females survived more compared to males  
- 1st Class passengers had better survival rates than 3rd Class  
- Most passengers embarked from Southampton, but survival was lower compared to Cherbourg
